# TaTeTi (Tres en línea) y Sistema de Tickets - Proyecto en Python con Tkinter

Este proyecto combina un juego de TaTeTi (Tres en línea) desarrollado en Python utilizando Tkinter, junto con un sistema de tickets básico para el manejo de incidencias.

## TaTeTi (Tres en línea)

### Características

-   Interfaz gráfica simple desarrollada con Tkinter.
-   Jugador humano contra una inteligencia artificial básica.
-   Reinicio del juego después de cada partida.
-   Indicación de ganador o empate.

### Requisitos

-   Python 3.x
-   Biblioteca estándar de Python: tkinter

### Instalación y Ejecución del Juego TaTeTi

1.  Clona o descarga este repositorio en tu máquina local:
    
    bash
    
    Copiar código
    
    `git clone https://github.com/tu_usuario/tateti-python-tkinter.git` 
    
2.  Asegúrate de tener Python 3.x instalado. Puedes descargar Python desde [python.org](https://www.python.org/downloads/).
    
3.  No se requieren dependencias adicionales más allá de las bibliotecas estándar de Python.
    
4.  Navega hasta el directorio donde clonaste o descargaste el repositorio.
    
5.  Ejecuta el script `tateti.py` para iniciar el juego:
    
    Copiar código
    
    `python tateti.py` 
    
6.  Se abrirá una ventana con el juego TaTeTi. Juega haciendo clic en los botones del tablero.
    
7.  Puedes reiniciar el juego en cualquier momento haciendo clic en el botón "Reiniciar juego".
    

## Sistema de Tickets

El sistema de tickets permite la gestión básica de incidencias mediante un sistema de archivos.

### Características

-   Creación de tickets con información de nombre, sector, asunto y problema.
-   Lectura de tickets existentes mediante su número de ticket.
-   Almacenamiento persistente en un archivo de texto.

### Requisitos

-   Python 3.x

### Instalación y Ejecución del Sistema de Tickets

1.  El archivo `tickets.py` maneja el sistema de tickets.
    
2.  Asegúrate de tener Python 3.x instalado. Puedes descargar Python desde [python.org](https://www.python.org/downloads/).
    
3.  No se requieren dependencias adicionales más allá de las bibliotecas estándar de Python.
    
4.  Navega hasta el directorio donde se encuentra el archivo `tickets.py`.
    
5.  Ejecuta el script `tickets.py` para iniciar el sistema de tickets:
    
    Copiar código
    
    `python tickets.py` 
    
6.  Se abrirá un menú principal donde puedes seleccionar entre alta de ticket, lectura de ticket o salir del programa.
    
7.  Sigue las instrucciones en pantalla para interactuar con el sistema de tickets.
